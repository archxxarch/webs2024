01. 변수 : 데이터 불러오기

데이터를 불러오는 변수는 프로그래밍 및 데이터 분석 작업에서 중요한 역할을 하는 변수입니다. 데이터를 처리하고 분석하기 위해 필요한 정보를 컴퓨터 메모리에 저장하고 관리하는데 사용됩니다. 

{
    let x = 100, y = 200, z = "javascript";

    console.log(x, y, z);
}

let 키워드를 사용하여 세 개의 변수 x, y, z를 선언하고 값을 할당합니다.
x 변수에는 100이라는 정수가 할당됩니다.
y 변수에는 200이라는 정수가 할당됩니다.
z 변수에는 "javascript"라는 문자열이 할당됩니다.

결과 확인하기
100
200
javascript

02. 상수 : 데이터 불러오기

상수는 변하지 않는 값을 나타내는 변수입니다. 데이터를 불러오는 과정에서 상수를 사용하면 코드를 더 명확하고 유지 관리하기 쉽게 만들 수 있습니다. 데이터 소스, 연결 정보, 설정 값 등을 상수로 정의하여 코드 내에서 일관성을 유지하고 오류를 방지할 수 있습니다. 

{
    const x = 100, y = 200, z = "javascript";

    console.log(x, y, z);
}

const 키워드를 사용하여 세 개의 상수 x, y, z를 선언하고 값을 할당합니다.
x 상수에는 100이라는 정수가 할당됩니다.
y 상수에는 200이라는 정수가 할당됩니다.
z 상수에는 "javascript"라는 문자열이 할당됩니다.

03. 배열 : 데이터 불러오기

배열은 JavaScript 및 다른 프로그래밍 언어에서 주로 사용되는 데이터 구조 중 하나로, 여러 값을 하나의 변수에 저장하고 관리하는 데 사용됩니다. 배열에서 특정 위치의 데이터를 불러오려면 인덱스를 사용합니다. 인덱스는 0부터 시작하며, [인덱스]를 사용하여 요소에 접근합니다. 

{
    const arr = [100, 200, "javascript"];

    console.log(arr[0], arr[1], arr[2]);
}

console.log() 함수를 사용하여 배열 arr의 각 요소를 콘솔에 출력합니다.
arr[0]는 배열의 첫 번째 요소를 나타내며, 해당 요소는 숫자 100입니다.
arr[1]는 배열의 두 번째 요소를 나타내며, 해당 요소는 숫자 200입니다.
arr[2]는 배열의 세 번째 요소를 나타내며, 해당 요소는 문자열 "javascript"입니다.

결과 확인하기
100
200

04. 배열 : 데이터 갯수 불러오기

데이터의 갯수 또는 배열의 길이를 확인하는 방법은 배열의 length 속성을 사용하는 것입니다. 배열의 length 속성은 배열에 포함된 요소의 개수를 나타냅니다. 데이터의 갯수 또는 배열의 길이를 확인하려면 배열 이름 뒤에 .length를 붙여 사용하면 됩니다. 이를 통해 배열 내 요소의 개수를 얻고, 이를 활용하여 배열을 반복하거나 조작하는 등의 작업을 수행할 수 있습니다. 

{
    const arr = [100, 200, "javascript"];

    console.log(arr.length);
}

console.log() 함수를 사용하여 배열 arr의 length 속성을 출력합니다.
배열의 length 속성은 배열에 포함된 요소의 개수를 나타내는 속성입니다.

05. 배열 : 데이터 불러오기 : for()

for문의 일반적인 구조는 for(초깃값; 조건식; 증감값){}입니다.
초깃값은 반복문이 시작될 때 한 번만 실행되며, 반복문 변수를 초기화합니다. 주로 반복 변수를 선언하고 초기값을 설정하는 부분입니다.
조건식은 각 반복마다 검사되는 조건입니다. 조건이 참(true)인 동안 반복문이 계속 실행됩니다. 만약 조건이 거짓(false)이면 반복문이 종료됩니다.
증감값은 각 반복이 실행된 후 반복 변수를 어떻게 업데이트할지를 정의합니다. 주로 반복 변수를 증가(i++) 또는 감소(i--)시키는데 사용됩니다.
{}은 반복문의 몸체로, 조건이 참인 동안 반복적으로 실행되는 코드를 포함합니다. 

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    for(let i=0; i<9; i++)
    {
        console.log(arr[i]);
    }
}

const 키워드를 사용하여 arr이라는 이름의 배열을 선언하고 초기값으로 9개의 숫자를 가지는 배열을 생성합니다.
for 루프를 사용하여 배열의 요소를 반복합니다.
let i = 0: i라는 변수를 초기화하고 0으로 설정합니다. 이 변수는 배열의 인덱스를 나타내며, 초기값은 0입니다.
i < 9: 루프를 계속 실행할 조건을 설정합니다. i가 9보다 작을 때 루프가 계속됩니다.
i++: 루프가 한 번 실행될 때마다 i 값을 1씩 증가시킵니다.
console.log(arr[i]): 현재 인덱스 i에 해당하는 배열 arr의 요소를 콘솔에 출력합니다.
루프가 실행될 때 i는 0부터 8까지 증가하면서 배열의 요소를 하나씩 출력합니다.

06. 배열 : 데이터 불러오기 : forEach()

forEach문의 일반적인 구조는 arr.forEach(function(요소, 인덱스, 배열)) {} 입니다. 요소는 배열의 각 요소에 대한 참조, 인덱스는 요소의 인덱스(위치)를 나타내고, 배열은 forEach()를 호출한 배열 자체입니다.
forEach() 메서드를 활용하면 반복 작업을 더 간단하게 처리할 수 있으며, 코드의 가독성을 높이고 유지 보수를 쉽게 만들 수 있습니다. 이 메서드는 배열의 각 요소를 순회하면서 필요한 작업을 수행할 때 유용합니다. 

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    // 배열 요소 출력
    arr.forEach(function(element){
        console.log(element);
    });

    // 배열 인덱스 출력
    arr.forEach(function(element, index){
        console.log(index);
    });

    // 전체 배열 출력
    arr.forEach(function(element, index, array){
        console.log(array);
    });

    // 약식 표현
    arr.forEach((el) => {
        console.log(el);
    });

    arr.forEach(el => {
        console.log(el);
    });

    arr.forEach(el => console.log(el));
}

1. 배열 요소 출력 : 배열 arr의 각 요소를 순회하며 요소를 콘솔에 출력합니다.
2. 배열 인덱스 출력 : 배열 arr의 각 요소와 해당 요소의 인덱스를 순회하며 인덱스를 콘솔에 출력합니다.
3. 전체 배열 출력 : 배열 arr의 각 요소와 해당 요소의 인덱스 및 전체 배열을 순회하며 배열 전체를 콘솔에 출력합니다.
4. 약식 표현 : 화살표 함수를 사용하여 배열 arr의 각 요소를 순회하며 요소를 콘솔에 출력합니다.

07. 배열 : 데이터 불러오기 : for of

for of문의 기본 구조는 (변수 of 컬렉션) {}입니다. 변수는 각 반복에서 컬렉션의 요소가 할당되는 변수이고 컬렉션은 반복 가능한(iterable) 객체, 즉, 반복할 컬렉션입니다.
반복문을 직접 작성하는 것보다 코드를 간결하게 만들어주며, 배열 뿐만 아니라 문자열 등 다양한 반복 가능한 객체에 적용할 수 있습니다. 

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    for(let element of arr){
        console.log(element);
    }
}

arr이라는 배열을 선언하고 초기화합니다. 이 배열에는 숫자 요소가 포함되어 있습니다.
for...of 루프를 사용하여 arr 배열의 각 요소를 반복합니다. 이때, element라는 변수에 현재 요소가 할당됩니다.
console.log(element)를 사용하여 각 요소를 콘솔에 출력합니다.

08. 배열 : 데이터 불러오기 : for in

for in문의 기본 구조는 for (변수 in 객체) {} 입니다. 변수는 각 반복에서 속성 이름(키)이 할당되는 변수이고 객체는 반복할 객체입니다.
for in문은 객체의 속성을 순회할 때 유용하지만, 배열에 대해서는 사용하지 않는 것이 좋습니다. 배열의 인덱스 대신에 객체의 속성을 열거하므로 배열에 대한 반복에는 for of 문이 더 적합합니다. for in문을 사용할 때 주의해야 할 점은, 상속된 속성들도 함께 열거된다는 점입니다. 때문에 객체의 속성을 순회할 때, hasOwnProperty() 메서드를 사용하여 속성이 직접 객체에 속해 있는지 확인하는 것이 일반적으로 권장되는 방법입니다. 

{
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    for(let element in arr){
        console.log(arr[element]);
    }
}

for in문은 원래 객체에서 사용하지만, 배열에서는 element로 입력하면 인덱스 출력이 가능합니다.

09. 배열 : 데이터 불러오기 : map()

map() 메서드는 배열의 각 요소에 대해 주어진 함수를 호출하고 그 결과를 새로운 배열로 반환합니다. 원본 배열은 변경되지 않습니다. 

{
    const arr = [100, 200, "javascript"];

    arr.map(function(el, index, array){
        console.log("09 : " + el + " : " + index + " : " + array);
    });

    const arrNum = arr.map((el) => {
        return el;
    }); 

    console.log(arrNum);

    console.log("09 : " + arrNum);
}

const arrNum = arr.map((el) ) : map 메서드를 사용하여 arr 배열의 각 요소를 변환하고 새로운 배열 arrNum에 저장합니다. 화살표 함수를 사용하여 간단하게 요소를 그대로 반환하는 콜백 함수를 정의합니다. 따라서 arrNum은 arr과 동일한 요소를 가진 새로운 배열이 됩니다.
console.log(arrNum); : arrNum을 출력하여 새로운 배열의 내용을 확인합니다. 따라서 arrNum에는 [100, 200, "javascript"] 배열이 저장되어 있으므로 이 배열이 출력됩니다.
console.log("09 : " + arrNum); : arrNum은 배열이므로 이어진 요소들을 쉼표와 공백으로 구분하여 출력됩니다.

결과 확인하기
[100, 200, "javascript"]
09 : 100, 200, javascript

10. 배열 : 데이터 불러오기 : filter()

filter() 메서드는 배열 내 각 요소에 대해 주어진 조건 함수를 호출하고, 조건 함수가 true를 반환하는 요소들로 이루어진 새로운 배열을 반환합니다. 

{
    const arr = [100, 200, "javascript"];

    arr.filter(function(el, index, array){
        console.log("10 : " + el + " : " + index + " : " + array);
    });

    const arrNum = arr.filter((el) => {
        return el > 150;
    });

    // 약식 표현
    // const arrNum2 = arr.filter(el => {
    //     return el > 150;
    // });

    // const arrNum3 = arr.filter(el => el > 150);

    console.log(arrNum);
}

const arrNum = arr.filter((el) => { return el > 150;}); : filter 메서드를 사용하여 arr 배열에서 조건을 만족하는 요소만 추출하여 arrNum에 저장합니다. 화살표 함수를 사용하여 각 요소가 150보다 큰지를 검사하고, 조건을 만족하는 경우에만 새 배열에 포함됩니다.
console.log(arrNum); : arrNum을 출력하여, 조건을 만족하는 요소만을 포함하는 새 배열을 확인합니다. 따라서 arrNum에는 [200]이라는 배열이 저장되어 있으므로 이 배열이 출력됩니다. 150보다 큰 요소만 남겼기 때문에 100은 제외됩니다.

11. 배열 : 데이터 불러오기 : 배열 펼침 연산자

배열 펼침 연산자(Spread Operator)는 JavaScript에서 배열을 확장하거나 복사할 때 사용되는 편리한 구문입니다. 

{
    // 배열 펼침 연산
    const arr1 = [100, 200, 300];
    const arr2 = [400, 500, 600];

    console.log(arr1);               
    console.log(...arr1);              
    console.log(arr1 + arr2);         
    console.log(...arr1, ...arr2);    
    console.log(...arr1, 400);         
    console.log(...arr1.slice(0,2)); 

    // 함수 호출시 인수 전달
    function addNum(a, b, c){
        return a + b + c;
    }
    const num = [100, 200, 300];
    const sum = addNum(...num);
    console.log(num);
}

console.log(...arr1.slice(0,2)); : slice(0, 2) 메서드를 사용하여 arr1 배열의 처음 두 개 요소를 추출한 후, 펼침 연산자를 사용하여 출력합니다.
함수 호출시 인수 전달 : addNum 함수에 배열 num의 요소를 펼쳐서 전달하고 함수에서 이 요소들을 사용하여 계산한 결과를 출력합니다. 그러나 num 배열 자체는 변경되지 않으며 그대로 유지됩니다.

12. 배열 : 데이터 불러오기 : 배열 구조 분해 할당

배열 구조 분해 할당(Array Destructuring Assignment)은 JavaScript에서 배열의 요소를 추출하고 할당하는 방법을 제공합니다. 이를 통해 배열에서 원하는 요소를 쉽게 추출하고 변수에 할당할 수 있습니다. 

{
    // 단일 배열에서 구조 분해 할당
    const [a, b, c] = [100, 200, 300];

    console.log(a);
    console.log(b);
    console.log(c);

    // 기존 배열에서 구조 분해 할당
    const arr = [100, 200, 300];
    const [x, y, z] = arr;

    console.log(x);
    console.log(y);
    console.log(z);

    // 나머지 요소 추출 분해 할당
    const num = [100, 200, 300, 400, 500];
    const [num1, num2, ...rest] = num;

    console.log(num1);
    console.log(num2);
    console.log(rest);         
    console.log(...rest);      
}

나머지 요소 추출 분해 할당 : [num1, num2, ...rest]를 사용하여 배열 num의 처음 두 요소를 num1과 num2 변수에 할당하고, 나머지 요소는 rest 배열로 추출합니다. ...rest를 사용하여 rest 배열의 요소를 펼쳐서 출력할 수도 있습니다.
