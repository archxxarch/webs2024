01. 변수 : 데이터 저장

자바스크립트(JavaScript)에서 데이터를 저장하는 저장소입니다.

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    console.log(x);
    console.log(y);
    console.log(z);
}

var: 과거에 사용되던 변수 선언 방식으로, 현재는 잘 사용하지 않습니다.
var 재할당 가능
let, const 재할당 불가능
const 변경 불가능 -> 에러
문자 + 숫자 -> 숫자

03. 변수 : 데이터 저장 + 데이터 변경 + 데이터 추가

x += x + 100 은 x = x + 100 을 의미합니다.

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    x += 100;
    y += 100;    
    z += "jquery";    

    console.log(x);
    console.log(y);
    console.log(z);
}

console.log(x); 변수 x의 값을 출력합니다. 출력 결과: 200 (원래 값 100에 100을 더함)
console.log(y); 변수 y의 값을 출력합니다. 출력 결과: 300 (원래 값 200에 100을 더함)
console.log(z); 변수 z의 값을 출력합니다. 출력 결과: "javascriptjquery" (원래 값 "javascript"에 문자열 "jquery"를 덧붙임)

04. 상수 : 데이터 저장 + 데이터 변경(X)

상수(Constant)는 프로그래밍에서 값을 한 번 할당하면 그 값을 변경할 수 없는 변수를 말합니다. 즉, 상수는 한 번 값이 할당되면 프로그램의 실행 도중에 변경할 수 없는 고정된 값입니다. 

{
    const x = 100;
    const y = 200;
    const z = "javascript";

    console.log(x);
    console.log(y);
    console.log(z);
}

const: 변수를 선언할 때 사용하는 키워드 중 하나로, 상수(constant)를 정의하는 데 사용됩니다. const로 선언된 변수는 한 번 값이 할당되면 그 이후에 값을 변경할 수 없습니다. 이는 변수가 불변성을 가지며, 상수로서 사용될 때 유용합니다.

05. 배열 : 데이터 저장(여러개) : 표현방법1 : 배열 선언

배열은 여러 개의 값을 하나의 변수에 저장하고 관리하는 데 사용되는 데이터 구조입니다.

{
    const arr = new Array();

    arr[0] = 100;

07. 배열 : 데이터 저장(여러개) : 표현방법3 : 리터럴 방식으로 배열 선언

리터럴 방식으로 배열을 선언하는 것은 가장 간단하고 흔한 방법 중 하나입니다. 배열 리터럴은 대괄호 [ ]를 사용하여 배열을 생성하고 초기화하는 방식을 말합니다. 배열 내부에 요소들을 나열하여 배열을 초기화하게 됩니다.

09. 배열 : 데이터 저장(여러개) : 표현방법5 : 배열 구조 분해 할당

배열 구조 분해 할당(Array Destructuring Assignment)은 배열의 요소를 추출하여 변수에 할당하는 방법입니다. 이를 통해 배열의 각 요소를 더 쉽게 접근하고 활용할 수 있습니다. 

{
    const arr = [100, [200,300], ["javscript", "jquery"]];

    console.log(arr[0]);
    console.log(arr[1][0]);
    console.log(arr[1][1]);
    console.log(arr[2][0]);
    console.log(arr[2][1]);
}

console.log(arr[0]); 배열 arr의 첫 번째 원소를 콘솔에 출력합니다. 따라서 프로그램을 실행하면 100이 출력됩니다.
console.log(arr[1][0]); 배열 arr의 두 번째 원소는 또 다른 배열이기 때문에, 이중 배열 구조로 접근합니다. arr[1]은 [200, 300] 배열을 가리키고, 그 중 첫 번째 원소는 200이므로 콘솔에 200이 출력됩니다.
console.log(arr[1][1]); 위와 마찬가지로, 배열 arr의 두 번째 원소인 [200, 300] 배열에서 두 번째 원소를 출력합니다. 따라서 300이 콘솔에 출력됩니다.
console.log(arr[2][0]); 배열 arr의 세 번째 원소는 또 다른 배열 ["javascript", "jquery"]이므로, 이중 배열 구조로 접근하여 첫 번째 원소를 출력합니다. 프로그램을 실행하면 "javascript"가 출력됩니다.
console.log(arr[2][1]); 배열 arr의 세 번째 원소인 ["javascript", "jquery"]에서 두 번째 원소를 출력합니다. 따라서 "jquery"가 콘솔에 출력됩니다.

11. 배열 : 데이터 저장(여러개) : 표현방법6 : 배열 펼침연산자

배열 펼침 연산자(Spread Operator)는 배열 내의 요소들을 하나씩 펼쳐서 새로운 배열이나 함수 호출 등에서 사용할 수 있게 해주는 연산자입니다. 이 연산자는 ... (세 개의 점)으로 표현되며, 여러 상황에서 유용하게 활용됩니다.

{
    const arr1 = [100, 200, "javascript"];
    const arr2 = [300, 400, "jquery"];
    const arr3 = [...arr1, 500];

    console.log(arr1);             
    console.log(...arr1);         
    console.log(...arr1, ...arr2);  
    console.log(...arr3);             
}

console.log(...arr1); 배열 arr1을 전개 연산자로 펼쳐서 배열의 각 원소를 개별적으로 출력합니다. 실행하면 100 200 javascript가 출력됩니다.
console.log(...arr1, ...arr2); ...arr1과 ...arr2를 전개 연산자로 펼쳐서 두 배열의 모든 원소를 개별적으로 출력합니다. 실행하면 100 200 javascript 300 400 jquery가 출력됩니다.
console.log(...arr3); 배열 arr3을 전개 연산자로 펼쳐서 모든 원소를 개별적으로 출력합니다. 실행하면 100 200 javascript 500가 출력됩니다.

12. 객체 : 데이터 저장(키와값) : 표현방법1 : 객체 선언 후 배열방식으로 데이터 입력

객체의 프로퍼티는 키(key)와 값(value)으로 구성되며, 객체의 중괄호 내에 키와 값을 지정하여 데이터를 입력할 수 있습니다. 배열 방식으로 데이터를 입력하는 것은 특히 동적인 상황에서 유용합니다.

{
    const obj = new Object();

    obj[0] = 100;
    obj[1] = 200;
    obj[2] = "javascript";

    console.log(obj[0]);     
    console.log(obj[1]);      
    console.log(obj[2]);        
}

obj[0] = 객체 obj의 프로퍼티 이름으로 0을 사용하여 값을 할당합니다. 이렇게 하면 객체의 프로퍼티 0에 100이 할당됩니다.
obj[1] = 200; 객체 obj의 프로퍼티 이름으로 1을 사용하여 값을 할당합니다. 프로퍼티 1에 200이 할당됩니다.
obj[2] = "javascript"; 객체 obj의 프로퍼티 이름으로 2를 사용하여 문자열 "javascript"을 할당합니다.
console.log(obj[0]); 객체 obj의 프로퍼티 0의 값을 출력합니다. 실행하면 100이 출력됩니다.
console.log(obj[1]); 객체 obj의 프로퍼티 1의 값을 출력합니다. 실행하면 200이 출력됩니다.
console.log(obj[2]); 객체 obj의 프로퍼티 2의 값을 출력합니다. 실행하면 "javascript"가 출력됩니다.

01. 변수 : 지역변수 + 전역변수

지역 변수는 함수 내부에서 선언된 변수로, 해당 함수 내부에서만 접근할 수 있는 변수입니다. 전역 변수는 어떤 함수의 스코프 내에서도 선언되지 않은 변수로, 스크립트 전체에서 접근 가능한 변수입니다.

{
    let x = 100;                //전역변수
    let y = 200;                //전역변수
    let z = "javascript";       //전역변수

    console.log("함수밖1:" + x, y, z);

    function func(){
        let x = 300;             //지역변수
        y += 400;                //전역변수 y의 값에 400을 추가
        let z = "jquery";        //전역변수 z의 값에 문자열 "jquery"를 추가

        console.log("함수안:" + x, y, z);
    }
    func();

    console.log("함수밖2:" + x, y, z);
}

console.log("함수밖1:" + x, y, z); 전역 변수 x, y, z의 값을 출력합니다. 실행하면 함수밖1: 100 200 javascript이 출력됩니다.
console.log("함수안:" + x, y, z); 지역 변수 x, 전역 변수 y, 지역 변수 z의 값을 출력합니다.실행하면 함수밖1: 300 600 jquery이 출력됩니다.
console.log("함수밖2:" + x, y, z); 전역 변수 x, y, z의 값을 출력합니다. 실행하면 함수밖1: 100 600 javascript이 출력됩니다.

결과 확인하기

함수안: 300 600 jquery
함수밖2: 100 600 javascript

02. 산술연산자

산술연산자는 더하기(+), 빼기(-), 곱하기(*), 나누기(/), 나머지(%)가 있습니다. 연산자들은 숫자형 데이터에 적용되며, 기본적인 산술 연산을 수행하거나 값의 증감을 조작하는 데 사용됩니다.

{
    let x = 4
    let y = 20

    console.log(x+y);
    console.log(x-y);
    console.log(x*y);
    console.log(x/y);
    console.log(x%y);
}
산술 연산자의 종류와 기본형
종류 기본형 설명
+ A+B 더하기
- A-B 빼기
* A*B 곱하기
/ A/B 나누기
% A%B 나머지

console.log(x+y); 변수 x와 y의 합을 출력합니다. 실행하면 24가 출력됩니다.
console.log(x-y); 변수 x와 y의 차를 출력합니다.실행하면 -16가 출력됩니다.
console.log(x*y); 변수 x와 y의 곱을 출력합니다.실행하면 80가 출력됩니다.
console.log(x/y); 변수 x를 변수 y로 나눈 결과를 출력합니다.실행하면 0.2가 출력됩니다.
console.log(x%y); 변수 x를 변수 y로 나눈 나머지를 출력합니다.실행하면 4가 출력됩니다.

03. 문자 결합 연산자

04. 대입 연산자

대입 연산자는 변수에 값을 할당하는 데 사용되는 연산자입니다. 대입 연산자는 다른 연산자와 함께 사용하여 연산 결과를 변수에 저장하는 데에도 사용될 수 있습니다. 이를 통해 변수 값을 갱신하거나 계산 결과를 보다 간결하게 처리할 수 있습니다. 이런 종류의 연산자를 복합 대입 연산자라고 합니다.

{
    let x = 18
    let y = 2

    console.log(x+=y);  // x = x + y
    console.log(x-=y);  // x = x - y
    console.log(x*=y);  // x = x * y
    console.log(x/=y);  // x = x / y
    console.log(x%=y);  // x = x % y
}

console.log(x+=y); 변수 x에 변수 y의 값을 더한 후, 그 결과를 변수 x에 할당하고 출력합니다. 실행하면 20이 출력됩니다.
console.log(x-=y); 변수 x에서 변수 y의 값을 뺀 후, 그 결과를 변수 x에 할당하고 출력합니다. 실행하면 18이 출력됩니다.
console.log(x*=y); 변수 x에 변수 y의 값을 곱한 후, 그 결과를 변수 x에 할당하고 출력합니다. 실행하면 36이 출력됩니다.
console.log(x/=y); 변수 x를 변수 y의 값으로 나눈 후, 그 결과를 변수 x에 할당하고 출력합니다. 실행하면 18이 출력됩니다.
console.log(x%=y); 변수 x를 변수 y의 값으로 나눈 나머지를 구한 후, 그 결과를 변수 x에 할당하고 출력합니다. 실행하면 0이 출력됩니다.

05. 증감 연산자

증감 연산자는 변수의 값을 1씩 증가시키거나 감소시키는 데 사용되는 연산자입니다. 증감 연산자는 변수 앞에 붙일 수도 있고, 변수 뒤에 붙일 수도 있습니다. 주로 반복문이나 변수 값의 조작에 활용됩니다.

{
    let num1 = 10;  
    let num2 = 20;  

    num1--;              // num1을 1 감소
    console.log(num1);   // 감소된 num1 출력

    num1++;              // num1을 1 증가
    console.log(num1);   // 증가된 num1 출력

    result=num2++;       // 후위 증가: num2의 현재 값(20)을 result에 할당, 그 후 num2를 1 증가
    console.log(result);

    result=++num2;        // 선위 증가: num2를 1 증가한 후의 값(22)을 result에 할당
    console.log(result);
}

후위 증가 연산자 (x++): 변수의 값을 먼저 사용한 후, 그 다음에 1을 증가시킵니다.
선위 증가 연산자 (++x): 변수의 값을 1 증가시킨 후, 그 다음에 증가된 값을 사용합니다.
감소 연산자도 마찬가지로 후위 감소 연산자(x--)와 선위 감소 연산자(--x)로 사용될 수 있습니다.

07. 논리 연산자

논리 연산자를 사용하여 여러 개의 조건을 결합하거나 부정하여 논리적인 조건을 평가합니다.

{
    let x = true;
    let b = false;

    console.log(a||b);      // 논리 OR 연산: x 또는 b가 true인지 확인
    console.log(a&&b);      // 논리 AND 연산: x와 b 모두 true인지 확인
    console.log(!b);        // 논리 NOT 연산: b의 반대값인 true로 변환
}

논리 AND(&&)는양쪽 조건이 모두 참일 때만 전체 표현식이 참이 됩니다.
논리 OR(||)은 양쪽 조건 중 하나 이상이 참이면 전체 표현식이 참이 됩니다.
논리 NOT(!)은 조건을 부정하여 반대의 결과를 얻습니다.

09. 삼항 조건 연산자

삼항 조건 연산자는 조건에 따라 두 가지 값 중 하나를 선택하는 데 사용되는 간단한 연산자입니다. 

{
    let a = 10;
    let b = 20;
    let c = a > b ? "javascript" : "hello"

    console.log(c);
}

let c = a > b ? "javascript" : "hello"; 조건 연산자를 사용하여 변수 a와 b를 비교합니다. 만약 a가 b보다 크다면 "javascript"을, 그렇지 않으면 "hello"를 변수 c에 할당합니다. 조건 a > b가 false이므로 "hello"가 변수 c에 할당됩니다.
