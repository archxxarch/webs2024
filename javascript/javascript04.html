01. if문

if문은 조건식을 만족(true)할 경우에만 코드를 실행합니다. else문은 조건식을 만족할경우(true)와 만족하지 않을 경우(false)에 따라 실행되는 코드가 달라집니다. 즉, 두가지 결과가 나올 수 있습니다. 

{
    if(null){
        document.write("실행되었습니다.(true)");
    } else {
        document.write("실행되었습니다.(false)");
    }
}

true, "문자열", 1, 2, "1", "2", [], {} 은 true를 출력하고, false, 0, null, undefined, ""(빈문자열)은 false를 출력합니다.

02. 다중 if문

else if문은 두가지 이상의 조건식과 정해 놓은 조건을 만족하지 않았ㅇ르 때 실행되는 코드로 이루어져 있습니다. 

{
    const num = 100;

    if(num == 90){
        document.write("실행되었습니다.(num = 90)");
    } else if(num == 95){
        document.write("실행되었습니다.(num = 95)")
    } else if(num == 100){
        document.write("실행되었습니다.(num = 100)")
    } else if(num == 105){
        document.write("실행되었습니다.(num = 105)")
    } else {
        document.write("실행되었습니다.");
    }
}

num 변수에 할당된 값이 100이므로, 두 번째 else if 조건인 num == 100이 참(True)으로 평가됩니다. 다른 조건은 모두 거짓(False)이므로 해당 조건에 따른 코드는 실행되지 않습니다.

03. 중첩 if문

중첩 if문은 조건문 안에 조건문이 있는 형태입니다. 

{
    const num = 100;

    if(num == 100){
        document.write("실행되었습니다.1");
        if(num == 100){
            document.write("실행되었습니다.2");
            if(num == 100){
                document.write("실행되었습니다.3");
            }
        }
    } else {
        document.write("실행되었습니다.4");
    }
}

num 변수가 100인 경우에만 모든 중첩된 if 문이 참(True)으로 평가되어 실행됩니다.만약 num의 값이 100이 아니라면 else 블록의 코드가 실행될 것입니다.

04. if문 생략 & 삼항 연산자

삼항 조건 연산자는 프로그래밍에서 사용되는 조건부 표현 방식 중 하나로, 조건에 따라 두 가지 다른 값을 반환하는 역할을 합니다. 이 연산자는 조건문(if-else)을 간단하게 표현할 수 있습니다. 

{
    const num = 100;

    // if(num){
    //     document.write("실행되었습니다.(true)");
    // }

    if(num == 100) document.write("실행되었습니다.(true)");

    // if(num){
    //     document.write("실행되었습니다.(true)");
    // } else {
    //     document.write("실행되었습니다.(false)");
    // }

    if(num == 100) document.write("실행되었습니다.(true)");
    else document.write("실행되었습니다.(false)");

    // 삼항 연산자(조건식 연산자)
    (num == 100) ? document.write("true") : document.write("false");
}

if(num == 100) document.write("실행되었습니다.(true)"); 이 코드 블록은 num 변수가 100과 같은지 검사합니다.조건이 참이면 "실행되었습니다.(true)"를 출력합니다.
if(num == 100) document.write("실행되었습니다.(true)"); else document.write("실행되었습니다.(false)"); 이 코드 블록은 num 변수가 100과 같은지 검사합니다. 조건이 참이면 "실행되었습니다.(true)"를 출력하고, 그렇지 않으면 "실행되었습니다.(false)"를 출력합니다.
(num == 100) ? document.write("true") : document.write("false"); 삼항 조건 연산자를 사용하여 num 변수가 100과 같은지 검사합니다. 조건이 참이면 "true"를 출력하고, 그렇지 않으면 "false"를 출력합니다.

05. switch문

선택문이 switch문은 변수에 저장된 값과 switch문에 있는 경우(case)의 값을 검사하여 변수와 경우의 값에서 일치하는 값이 있을떄 그에 해당하는 코드를 실행합니다. if문과 용도는 비슷하나 if문은 만족하는 데이터가 여러 개일 경우에 주로 사용하고, switch문은 여러 경우의 값 중 일치하는 데이터를 찾아 그에 해당하는 코드를 실행시킬 때 사용합니다. 

{
    const num = 100;

    switch(num){
        case 90:
            document.write("실행되었습니다.(90)");
            break;
        case 95:
        document.write("실행되었습니다.(95)");
            break;
        case 100:
        document.write("실행되었습니다.(100)");
            break;
        case 105:
        document.write("실행되었습니다.(105)");
            break;
        default:
            document.write("실행되었습니다.");
    }
}

switch 문은 각 case 레이블에 해당하는 값과 num 변수의 값이 일치하는 경우에 실행됩니다. 만약 어떤 case 레이블과도 일치하지 않는다면 default 레이블의 코드가 실행됩니다.

06. while문

while문은 조건식을 만족할 때까지 코드를 여러회 반복하여 실행할 수 있습니다. 

{
    let num = 1;
    while(num<=10){
        document.write(num);
        num++;
    }
}

while 루프를 사용하여 num 변수가 1부터 10까지의 값을 출력하는 예제입니다. while 루프는 주어진 조건이 참(True)인 동안 코드 블록을 반복 실행합니다. num 변수는 초기값으로 1을 가지고 있으며, while 루프의 조건은 num이 10 이하일 때를 나타내는 num <= 10입니다. 따라서 num이 10 이하인 동안에는 document.write(num)을 실행하고, 그 후에 num을 1씩 증가시킵니다.

07. do while문

while문의 경우에는 조건식의 만족 여부를 먼저 검사한 후 중괄호에 있는 코드의 실행 여부를 결정했습니다. 하지만 do while문은 반드시 한번은 코드를 실행하고 조건식을 검사하므로 조건이 처음부터 거짓이더라도 코드 블록이 한 번 실행됩니다. 

{
    let num = 1;
    do {
        document.write(num);
        num++;
    } while(num<=10);
}

num 변수는 초기값으로 1을 가지고 있으며, do-while 루프의 조건은 num이 10 이하일 때를 나타내는 num <= 10입니다. 따라서 do 블록 안의 코드가 실행되고, 그 후에 num을 1씩 증가시킵니다. 그런 다음 while 조건을 확인하고, num이 10 이하인 동안에는 계속 반복 실행됩니다.

08. for문

for문은 조건식을 만족할 때까지 특정코드를 반복하여 실행합니다. 사용 방법은 while문과 같지만 while문보다 사용하기 편하고 반복 횟수와 초기화, 증감 등을 한눈에 파악할 수 있기 때문에 반복 작업을 처리할 때 사용빈도가 높은 편입니다. 

{
    for(let i=1; i<=10; i++){
        document.write(i);
        }

}

let i = 1;: 루프가 시작될 때 i 변수를 1로 초기화합니다. i <= 10;: 루프의 조건은 i가 10 이하인 동안 반복을 진행합니다. i++: 각 반복 후에 i 값을 1씩 증가시킵니다.

09. break문

반복문인 while문 또는 for문에서 break문을 실행하면 조건식과 상관없이 강제로 반복문을 종료합니다. 즉, break문은 반복문을 강제로 종료할 때 사용합니다. 

{
    for(let i=1; i<10; i++){
        if( i == 5 ){
            break;
        }
    }
    document.write(i);
}

if 문 내에서 i 값이 5와 같을 때 break 문을 사용하여 루프를 종료하도록 조건을 설정하고 있습니다. 결과적으로 i가 5일 때 루프가 종료되므로, document.write(i) 코드에서는 i의 최종 값인 5가 출력됩니다.

10. continue문

continue문은 반복문에서만 사용할 수 있습니다. while문에 사용할 경우 continue문 다음애 오는 코드는 무시하고 바로 조건식으로 이동해 조건 검사를 합니다. 

{
    for(let i=1; i<10; i++){
        if( i == 5 ){
            continue;
        }
    }
    document.write(i);
}

for 루프를 사용하여 i가 1부터 10보다 작은 값인 동안 반복하고 있습니다. 그리고 if 문 내에서 i 값이 5와 같을 때 continue 문을 사용하여 현재 반복을 종료하고 다음 반복으로 넘어가도록 설정하고 있습니다. 주의할 점은 continue 문이 실행되면 현재 반복이 종료되고 다음 반복이 시작되지만, i의 값은 여전히 증가합니다. 따라서 i가 5와 같을 때 현재 반복을 종료하고 6으로 증가한 후 다음 반복이 시작됩니다. 결과적으로 for 루프에서 i가 1부터 9까지 반복하고, 그 후에 i가 10이 되어 루프가 종료됩니다. 마지막으로 document.write(i) 코드에서는 i의 값 10이 출력됩니다.
