01. 1부터 10까지 출력하기(초깃값0)

for()문을 이용하여 1부터 10까지 출력하는 예제입니다.

{
        for(let i=0; i<10; i++){
            console.log(i+1);
        }
}

2. for 키워드는 반복문을 시작하는 키워드입니다.
(let i = 0; i < 10; i++): for 루프의 세 가지 부분으로 구성됩니다.
let i = 0: 반복을 시작할 때 i라는 변수를 0으로 초기화합니다. 이 변수는 반복 동안 계속 증가하거나 감소합니다.
i < 10: 조건 부분입니다. i가 10보다 작을 때 반복이 계속됩니다. 만약 조건이 거짓이 되면 반복이 종료됩니다.
i++: 반복이 한 번 실행될 때마다 i 값을 1씩 증가시킵니다. 이 부분을 통해 반복이 언제 종료될지 결정합니다.
3. {}: 중괄호는 for 루프의 본문을 감싸는 역할을 합니다. 이 안에 반복될 코드를 작성합니다.
console.log(i+1): 현재 반복 중인 i+1 값을 출력하는 코드입니다. 결과적으로 1부터 10까지의 숫자가 순서대로 콘솔에 출력됩니다.

{

3. let table = "<table class='table'>";: table 변수를 선언하고, 표(테이블)를 시작하는 HTML 문자열을 할당합니다.
5. for (let i = 1; i <= 5; i++) : 바깥쪽 for 루프는 표의 각 행을 생성하는 역할을 합니다. 1부터 5까지의 값을 반복하며 각 행을 생성합니다.
6. table += "<tr>";: 각 행을 시작하는 <tr> 태그를 문자열에 추가합니다.
7. for (let j = 1; j <= 5; j++) : 안쪽 for 루프는 각 행당 5개의 열(셀)을 생성하는 역할을 합니다. 1부터 5까지의 값을 반복하며 각 셀을 생성합니다.
8. table += "<td>" + num + "</td>";: 각 셀에 출력될 HTML을 생성합니다. 현재의 num 값을 사용하여 각 셀에 짝수를 출력하고 있습니다. 이 값을 <td> 태그 안에 넣어 각 셀에 짝수가 표시됩니다.
9. num += 2;: num 변수에 2를 더하여 다음 셀에 출력될 숫자를 증가시킵니다. 이렇게 하면 각 셀에는 순차적으로 증가하는 짝수가 출력됩니다.
11. table += "</tr>";: 각 행을 닫는 </tr> 태그를 문자열에 추가하여 행을 종료합니다.
13. table += "</table>";: 모든 행과 열을 생성한 후, 표(테이블)를 닫는 </table> 태그를 문자열에 추가하여 표를 종료합니다.
14. document.write(table);: 최종적으로 생성된 표를 document.write() 함수를 사용하여 웹 페이지에 출력합니다.
